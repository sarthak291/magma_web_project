# magma_web_project
A responsive Magma website clone built using HTML, CSS, and JavaScript.
This project replicates the look and feel of the original Magma website, including modern UI sections, animations, images, and video elements.

Project Overview
The Magma Website Clone is a front-end project created to practice and demonstrate web development skills such as:
Responsive layouts
Modern UI design
Smooth animations
Multimedia integration (photos & videos)
This project focuses only on the frontend and does not include any backend functionality.

Technologies Used
HTML5 – Structure of the website
CSS3 – Styling, layout, animations, responsiveness
JavaScript – Interactivity and dynamic behavior
Images & Videos – For visual enhancement and realism

Features
✅ Fully responsive design
✅ Smooth scrolling and animations
✅ Image and video integration
✅ Modern UI inspired by Magma website
✅ Clean and organized code structure
